The Interface to the Veeqo API

## Installation

Add this line to your application's Gemfile:

```ruby
gem "veeqo", github: "abunashir/veeqo"
```

And then execute:

```sh
bundle install
```

### Setup

Clone the repository.

```sh
git clone https://github.com/abunashir/veeqo
```

Setup your environment.

```sh
bin/setup
```

Run the test suite

```sh
bin/rspec
```
